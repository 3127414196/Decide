发起连接：
msg{
	type:MSG_LOGIN,
	length:xxx
}
pMsg{
	"ip":"192.168.1.1",
	"port":111
}

gossip发送version:
msg{
	type:MSG_VERSION,
	length:xxx
}
pMsg{
	"version":xxx
}

收到version的结点发现自己的信息要更新时回复消息：
msg{
	type：MSG_REQUEST,
	length:xxx
}
pMsg{
	"version":xxx
}

发送结点发送数据：
msg{
	type:MSG_LIST,
	length:xxx
}
pMsg{
[
	{"key":"xxx",
	"value1":"xxx",
	"value2":"xxx"},
	{...},
	...
]
}
